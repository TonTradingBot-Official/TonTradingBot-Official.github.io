Welcome to TonTradingBot

TonTradingBot is your all-in-one solution for cryptocurrency trading developed on the TON blockchain, accessible directly through Telegram. It revolutionizes trading by offering a comprehensive suite of features designed to enhance trading efficiency and profitability.

Key Features

Multi-Level Referral System

TonTradingBot features a robust Referral System with five tiers, offering users the opportunity to earn lucrative rewards unlike any other bot on the market.

Join the Referral System
  • Level 1: Earn 300 points for each user you directly refer and 30% of their transaction fees.
  • Levels 2-5: Earn 50 points and 5% of transaction fees from trades made by referrals up to five levels deep.
  • Referral Statistics Display: Track your referral points, earnings, and more.
  • Claim Function: Make claims once every 24 hours, with a minimum claim amount of 0.5 TON.
  • View Referral Tree: See who referred each user and their trading volume.
  • User Check Function: Check if a user is in the system by entering their username.

Create and Use a Wallet or Multi-Wallet on TON

Before using any features of TonTradingBot, you have to set up your wallet with the Bot.

  • Create Wallet: Create a new wallet with a Wallet Address and 24 secret keywords.
  • Create X Wallet: Create up to 5 wallets at one time.
  • Import Wallet: Add an existing wallet using 24 secret keywords.
  • Wallet Checking: Check the wallet list with 24 secret keywords.
  • Delete Wallet: Delete any wallet on the TonTradingBot list.

Trading Tokens Using TonTradingBot

You can easily buy or sell tokens using TonTradingBot without using other wallet platforms.

  • Buy Transactions: Paste the Token Contract, check the token information, choose Buy Mode, and confirm the transaction.
  • Sell Transactions: Paste the Token Contract, check the token information, choose Sell Mode, and confirm the transaction.

Sniper Mode

The Sniper feature allows users to quickly buy newly listed tokens.

  • Creating a Sniper: Use Contract Address or Sniper Menu to set up sniping parameters.
  • Sniper Menu Features: List of active contracts, view contract details, add sniping wallet, and manage snipers.

Limit Order

Set buy or sell orders at specified prices using the Limit Orders feature.

  • Creating a Limit Order: Paste Contract, choose Add Limit Order, select mode (buy/sell), set the price and amount, and confirm.
  • Viewing Limit Orders: Monitor and manage limit orders from the Limit Orders panel.

Copy-Trading

Replicate trades from a source wallet to a destination wallet using the Copy Trade feature.

  • Set Up: Add Source Wallet, rename it if needed, and choose the destination wallet for copy-trading.
  • Configuring Copy Trade Settings: Toggle buy and sell functions, set default trade amounts, and activate Smart Mode.

TON Transfer

Transfer TON securely between wallets.

  • Steps: Select the wallet, specify the receiving wallet and amount, and confirm the transaction. Notifications for received TON can be toggled on or off.

Token Tracking

View and manage your token holdings with the Tracking feature.

  • Viewing Token List: Check Wallet Balance to view the top 10 tokens with a value greater than $1. Fast Swap allows quick access to the buy/sell interface for tokens.

Additional Features

  • Fast Swap: Displays recently traded tokens for quick selection.
  • New Listings: Scans and lists newly listed tokens meeting predefined standards.
  • Popular Tokens: Shows the most traded tokens and allows quick trades.
  • Honeypot Detection: Detects and prevents trading of scam tokens.
  • Multilingual Support: Supports multiple languages for a global audience.

Conclusion

TonTradingBot offers a comprehensive and user-friendly platform for cryptocurrency trading on the TON blockchain. Its integration with Telegram, coupled with a rich feature set, makes it an essential tool for traders aiming to optimize their strategies and enhance their market performance.
